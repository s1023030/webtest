Tips

git clone

git -b branch_name remote_link        // Clone the specific branch from remote repository

git commit

git commit --amend                    // to modify the comment of the last one commit

git push

git push --tag                        // Push tag to remote repository

git push -d origin tag_name            // Delete tag in remote repository

git branch

git branch                            // show all branch

git branch "branch_name"            // add new branch

git branch -d "branch_name"            // delete branch

git tag

Both branches and tags are essentially pointers to commits.
The big difference is that the commit a branch points to changes as you add new commits,
and a tag is frozen to a particular commit to mark a point in time as having a certain significance.
—–from stackoverflow

git tag                                //show all tag

git tag "tag_name" SHA-code            //stick the tag to commit

git tag -d 'tag_name'                //delete tag

git tag -l                            //List tags

git tag -l -n9                        //List tags with comments

The parameters for git describe

--tag                                // Get the latest git tag in repository

--long                                 // Get the latest git tag with long format

--always                             // If there is no tag in repository, it will return the SHA code of the latest commit.

The parameters for git log

--oneline                           //show information of each commits  in one line

--pretty=oneline                    //make them pretty >_。

--graph                             //show the graph of all commits

--grep "given_text"                 //to search the commit log for the given text:

--author "name"                     //Find the commit from someone

--S                                 //find all instances of a string in your repo's history, also which were deleted

--since

--until

--after

Fix conflict with binary files

git checkout --ours "file_name"

git checkout --theirs "file_name"

Remember to add and commit after checkout.
